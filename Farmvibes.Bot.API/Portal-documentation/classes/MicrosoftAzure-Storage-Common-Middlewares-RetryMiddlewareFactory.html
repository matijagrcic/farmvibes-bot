Create the retry handler for the Guzzle client, according to the given attributes.

public static create([string $type = self::GENERAL_RETRY_TYPE ][, int $numberOfRetries = Resources::DEFAULT_NUMBER_OF_RETRIES ][, int $interval = Resources::DEFAULT_RETRY_INTERVAL ][, string $accumulationMethod = self::LINEAR_INTERVAL_ACCUMULATION ][, bool $retryConnect = false ]) : RetryMiddleware
Parameters
$type : string = self::GENERAL_RETRY_TYPE

The type that controls the logic of the decider of the retry handler. Possible value can be self::GENERAL_RETRY_TYPE or self::APPEND_BLOB_RETRY_TYPE

$numberOfRetries : int = Resources::DEFAULT_NUMBER_OF_RETRIES

The maximum number of retries.

$interval : int = Resources::DEFAULT_RETRY_INTERVAL

The minimum interval between each retry

$accumulationMethod : string = self::LINEAR_INTERVAL_ACCUMULATION

If the interval increases linearly or exponentially. Possible value can be self::LINEAR_INTERVAL_ACCUMULATION or self::EXPONENTIAL_INTERVAL_ACCUMULATION

$retryConnect : bool = false

Whether to retry on connection failures.

Return values
RetryMiddleware —

A RetryMiddleware object that contains the logic of how the request should be handled after a response.
